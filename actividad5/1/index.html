Etiquetas "dl", "dt" y "dd".

Descripción: define una lista de descripción.

Esta etiqueta se usa junto con dt (define términos/nombres) y dd (describe cada término/nombre).

Ejemplo de uso de las etiquetas:

Cebolla
Es una verdura
Plátano
Es una fruta

Etiqueta "blockquote".

Descripción: especifica una sección que se cita de otra fuente.

Los navegadores generalmente aplican sangrían a los elementos.

Ejemplo de uso de la etiqueta:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation organization, WWF works in 100 countries and is supported by 1.2 million members in the United States and close to 5 million globally.

Etiqueta "q".

Descripción: define una cita corta.

Los navegadores normalmente insertan comillas alrededor de la cita.

Ejemplo de uso de la etiqueta:

WWF's goal is to: Build a future where people live in harmony with nature. We hope they succeed.

Etiqueta "cite".

define el título de una obra creativa (por ejemplo, un libro, un poema, una canción, una película, una pintura, una escultura, etc.).

El texto del elemento normalmente se muestra en cursiva.

Ejemplo de uso de la etiqueta:

The Scream by Edward Munch. Painted in 1893.

Etiqueta "abbr".

Descripción: define una abreviatura o un acrónimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM".

Ejemplo de uso de la etiqueta:

The WHO was founded in 1948.

Etiqueta "address".

Descripción: define la información de contacto del autor/propietario de un documento o artículo.

La información de contacto puede ser una dirección de correo electrónico, URL, dirección física, número de teléfono, identificador de redes sociales, etc.

El texto del elemento generalmente se presenta en cursiva y los navegadores siempre agregarán un salto de línea antes y después del elemento.

Ejemplo de uso de la etiqueta:

Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Etiqueta "sub".

Descripción: define el texto del subíndice. El texto del subíndice aparece medio carácter por debajo de la línea normal y, a veces, se representa en una fuente más pequeña. El texto de subíndice se puede utilizar para fórmulas químicas.

Ejemplo de uso de la etiqueta:

This text contains subscript text.

Etiqueta "sup".

Descripción: define el texto en superíndice. El texto en superíndice aparece medio carácter por encima de la línea normal y, a veces, se representa en una fuente más pequeña. El texto en superíndice se puede utilizar para notas al pie.

Ejemplo de uso de la etiqueta:

This text contains superscript text.

Etiqueta "code".

Descripción: se utiliza para definir una pieza de código de computadora. El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.

esta etiqueta no está obsoleta. Sin embargo, es posible lograr un efecto más rico usando CSS.

Ejemplo de uso de la etiqueta:

The HTML button tag defines a clickable button.

The CSS background-color property defines the background color of an element.

Etiqueta "pre".

Descripción: define texto preformateado.

El texto de un "pre" elemento se muestra en una fuente de ancho fijo y conserva tanto los espacios como los saltos de línea. El texto se mostrará exactamente como está escrito en el código fuente HTML.

Ejemplo de uso de la etiqueta:

    Text in a pre element
    is displayed in a fixed-width
    font, and it preserves
    both      spaces and
    line breaks

Etiqueta "var".

Descripción: se utiliza para definir una variable en programación o en una expresión matemática.

El contenido interior normalmente se muestra en cursiva.

Ejemplo de uso de la etiqueta:

The area of a triangle is: 1/2 x b x h, where b is the base, and h is the vertical height.

Etiqueta "kbd".

Descripción: se utiliza para definir la entrada del teclado.

El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.

Ejemplo de uso de la etiqueta:

Etiqueta "samp".

Descripción: se utiliza para definir la salida de muestra de un programa de computadora.

El contenido del interior se muestra en la fuente monoespaciada predeterminada del navegador.

Ejemplo de uso de la etiqueta:

Message from my computer:

File not found.
Press F1 to continue

Etiqueta "time".

Descripción: define una hora específica (o fecha y hora).

El "datetime" atributo de este elemento se usa para traducir la hora a un formato legible por máquina para que los navegadores puedan ofrecer agregar recordatorios de fechas a través del calendario del usuario y los motores de búsqueda puedan producir resultados de búsqueda más inteligentes.

Ejemplo de uso de la etiqueta:

Open from 10:00 to 21:00 every weekday.

I have a date on Valentines day.
